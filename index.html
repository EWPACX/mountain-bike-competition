四季联赛赛程安排

春季赛：2026年3月7-8日 | 夏季赛：2026年6月6-7日 | 秋季赛：2026年9月5-6日 | 冬季赛：2026年12月5-6日

四季联赛设有年度总积分榜，选手通过参加各赛季比赛积累积分，争夺年度总冠军荣誉及丰厚奖励。

第一章 总则
项目 内容
赛事宗旨 本次东大山地车公园 X 连吾体育四季联赛（春季赛）速降赛、土坡动作赛，以"推广山地自行车运动、搭建车友交流平台、传承极限运动精神、展现骑行竞技风采"为宗旨，秉持"公平、公正、公开、安全第一"的原则，为省内外山地车爱好者提供专业、规范、趣味的赛事体验，助力云南山地自行车运动持续发展，同时打造"四季有赛事、月月有活动"的山地骑行赛事IP
赛事名称 东大山地车公园 X 连吾体育 山地车四季联赛（春季赛）
赛事主题 悦动山野，四季骑缘
主办单位 东大山地车公园、连吾体育
承办单位 贵州连吾体育
场地支持 丘谷户外运动营地
比赛日期 2026年3月7日（星期六）至3月8日（星期日）
比赛地点 东大山地车公园（昆明市盘龙区东大村142号）
比赛项目 1. 山地车速降赛 (DH): 赛道全长约1000米，海拔落差约80米，采用"排位赛+决赛"制
2. 土坡技巧赛 (Dirt Jump): 裁判组根据动作难度、完成度、流畅性、风格综合评分
联赛说明 本赛事为"东大山地车公园 X 连吾体育四季联赛"的第一站（春季赛），全年共设春夏秋冬四站比赛。赛事设置年度积分系统，选手通过参加各赛季比赛积累积分，争夺年度总冠军荣誉及丰厚奖励。
注意： 如遇暴雨、雷电、地质灾害等极端天气或不可抗力，组委会有权推迟、缩短赛程、变更赛道或取消比赛。若取消，将根据实际情况办理部分退款或名额顺延。

第四章 参赛规则
项目 规则详情
参赛资格 1. 身体健康，具备相应骑行技能与体能
2. 完全了解比赛风险，自愿签署《免责声明书》
3. 18岁以下者必须由监护人签署文件和录制并全程陪同
4. 赛事组委会为所有选手购买比赛日基础意外险
报名时间 2026年3月5日24:00截止（各组别报满即止，逾期不再接受报名）
报名方式 通过主办方官方指定的线上平台报名，填写真实信息并选择组别
报名费用 速降赛： 138元/人
土坡技巧赛： 128元/人
双项目报名： 238元/人（优惠价）
费用包含：赛事包、号码牌、比赛日基础保险、参赛纪念品等。
取消与退款 报名成功后，因个人原因无法参赛：
1. 2026年2月28日18:00前申请，可退还50%报名费
2. 之后不予退款，但可按规定自行转让名额
竞赛规则 1. 比赛参照中国自行车运动协会审定的最新规则执行
2. 速降赛赛制： 排位赛+决赛，排位赛决定决赛发车顺序
3. 土坡赛赛制： 裁判从难度、完成度、流畅性、风格四个维度百分制评分
4. 公平竞赛： 选手须独立完成比赛，服从裁判判罚
5. 对判罚有异议，可于赛后30分钟内提交书面申诉并缴纳申诉费
安全与环保 1. 选手须始终将安全置于首位，遇身体不适立即停止比赛并求助
2. 赛事处于森林防火期，严禁在赛区内任何地方吸烟、生火野炊
3. 保护环境，所有垃圾必须带至指定丢弃点
4. 非参赛选手严禁进入赛道，禁止在赛道逆向推行或行走
名次录取 按章程中设定的"获奖名额"进行录取。若某组别实际完赛人数少于获奖名额，则按实际完赛人数递减录取
奖励办法 各组别前三名颁发奖杯、奖牌及证书，所有获奖选手将获得奖金或等值奖品。具体奖金分配方案将于赛前在官方平台公布
第五章 装备要求
装备类型 具体要求
车辆要求 基本要求：
1. 参赛自行车须为24-29英寸轮径的山地自行车
2. 必须使用宽度≥2.2英寸的齿胎
3. 推荐使用全避震车型，前叉避震行程建议≥100mm
4. 严禁使用带有金属螺钉或长钉的轮胎
5. 比赛前必须通过安全检查并粘贴合格标识
速降赛护具
(强制要求)		 1. 必须佩戴全盔
2. 必须佩戴护膝、护肘
3. 必须佩戴全指手套
4. 建议穿戴： 护甲、护颈等专业护具
5. 组委会或裁判有权根据天气或赛道状况，临时提升护具强制要求
土坡技巧赛护具
(强制要求)		 1. 必须佩戴全盔或高品质山地车半盔
2. 必须佩戴护膝、护肘
3. 建议穿戴： 全指手套、护甲等专业护具
通用禁令 1. 禁止佩戴玻璃镜片的眼镜
2. 不得穿拖鞋或赤脚骑行
3. 所有装备必须在赛前通过安全检查
装备检查 所有参赛车辆和护具必须在签到期间接受安全检查，合格后方可粘贴参赛标识。未通过检查或未粘贴标识的装备不得用于比赛。
安全须知： 装备要求是保障选手安全的基础，不符合要求的选手将被禁止参赛。请所有选手在赛前仔细检查自己的车辆和护具，确保符合章程要求。

四季联赛积分机制说明

积分计算规则

四季联赛采用积分制，选手在每站比赛中根据名次获得相应积分。全年四站比赛结束后，各组成绩最好的3站积分相加，得出年度总积分。

年度总积分与奖励

四季联赛结束后，各组别年度总积分前10名选手将获得"年度总冠军"、"年度亚军"、"年度季军"等荣誉称号及丰厚奖励。

年度总冠军奖励： 冠军奖杯 + 10000元现金/等价奖品 + 下赛季免费参赛资格 + 赞助商产品大礼包

年度亚军奖励： 亚军奖杯 + 5000元现金/等价奖品 + 下赛季半价参赛资格

年度季军奖励： 季军奖杯 + 3000元现金/等价奖品 + 下赛季8折参赛资格

积分规则说明：
  • 选手必须至少参加2站比赛才有资格参与年度排名
  • 积分计算取选手最好的3站成绩相加，允许一站成绩不理想
  • 如遇积分相同，比较选手最好单站成绩，再相同则比较第二好成绩，以此类推
  • 年度积分榜将在每站比赛结束后3个工作日内更新，并公示于官方网站
  • 积分系统最终解释权归东大山地车公园 X 连吾体育四季联赛组委会所有
